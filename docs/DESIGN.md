# Cordal Sur - Design System (awesomedesign.md)

## Brand Core
- **Vibe:** A modern mountain refuge. 50/50 balance between Winter (snow/ice) and Summer (forest/earth).
- **Keywords:** Mountain, Nature, Adventure, Warmth, Rest, Order, Security, Comfort, Premium, Snow, Forest.

## Color Palette
### ❄️ Winter (Base & Structure - Clean, crisp, airy)
- **Snow White:** `#F7FAFC` (Main background, clean spaces)
- **Ice Blue:** `#DDEAF3` (Subtle backgrounds, secondary cards)
- **Deep Blue:** `#12344D` (Primary text, structural headers)
- **Dark Night:** `#0B2538` (High contrast text, footers)

### 🌲 Summer (Accents & Warmth - Cozy, active, earthy)
- **Forest Green:** `#1F4D3A` (Primary buttons, success states, nature highlights)
- **Warm Wood:** `#B7793E` (Secondary buttons, warm accents, icons)
- **Volcanic Rock:** `#6B7280` (Muted text, secondary borders)
- **Copper/Fire:** `#C46A3A` (Alerts, urgent buttons, warmth)

## Typography
- **Primary Font:** `Inter`, sans-serif.
- **Headings:** Bold, tight letter-spacing (`-0.04em`), conveying a modern and premium feel.
- **Body:** Readable, high line-height (`1.6`), relaxed spacing for easy reading by guests.

## UI Components
- **Cards:** Glassmorphism effect. Background `rgba(255, 255, 255, 0.88)` with soft borders `1px solid rgba(255,255,255,0.82)`.
- **Shadows:** Soft and premium. `box-shadow: 0 12px 35px rgba(18, 52, 77, 0.10)`.
- **Border Radius:** Very rounded for a friendly, modern feel. Buttons (`999px`), Cards (`20px`), Containers (`28px`).
- **Gradients:** Subtle mesh gradients linking winter and summer (e.g., Deep Blue to Forest Green for icons).

## Layout Principles (Taste Constraints)
- **Mobile-First:** The primary user is a guest on their phone.
- **Breathing Room:** High padding (e.g., `22px` on cards, `42px` between sections).
- **No Clutter:** Avoid excessive icons or dense text. Use tags for quick info (e.g., [Huésped], [Invierno]).
- **Visual Balance:** Do not literally split the screen 50/50. Use white/ice backgrounds (Winter) accented with wood/green buttons and tags (Summer).
